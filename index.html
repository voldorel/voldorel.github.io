Hello!

Hello, I'm Mehdi Parvan, a 26-year-old Game Developer with a solid background in coding and a focus on C#. My journey into the world of programming began in 2013, with my professional career taking shape in 2019. .

I have a strong foundation in programming, including expertise in C#, but I also have experience in Java, C++, HTML, CSS, and JavaScript, which adds to my versatility.

Throughout my career, I've primarily worked on Online Games and VR games. In my most recent role, I started as a Unity Developer and was later promoted to Lead Unity Developer for the game "Mafia," which gained recognition in local Iranian markets such as Bazaar and Myket.

In addition to my professional roles, I've pursued freelance projects, from captivating VR experiences to innovative front-end web development, showcasing my adaptability and willingness to explore new horizons.

What distinguishes me is my keen interest in both the technical and artistic dimensions of game development. Apart from Unity, I've explored other game development tools like Game Maker Studio, complemented by Spine and Moho. I'm also proficient in 3D modeling using Blender and possess skills in traditional hand drawing and artistic expression.

Furthermore, I've delved into server development, having independently created an online game server using ASP.NET, which reflects my commitment to continuous learning and growth.

I'm well-versed in multiple programming languages, and I'm ready to contribute my skills and creativity to the dynamic field of game development, making a meaningful impact on every project I undertake.

Run and Gun Copy - Personal Project (2022)
Development Overview:
For this personal project, I embarked on a challenging endeavor to recreate a vertical slice of a renowned run-and-gun game entirely from scratch. The project aimed not only to replicate but to understand the intricacies of game design, programming, and modeling.
Key Accomplishments:
Vertical Slice Recreation: Recreating a vertical slice from the ground up involved meticulous attention to detail in both gameplay mechanics and visual elements.
Slo-Mo Event Integration: Implemented a dynamic slo-mo feature, adding an extra layer of complexity and engagement to the gameplay. Comprehensive Modeling in Blender: The entire game, from characters to environments, was meticulously modeled using Blender, emphasizing a cohesive and immersive visual experience.
Technical Insights:
The project provided invaluable insights into several key areas:
Game Design and Development: Understanding the nuances of game mechanics, from character movement to weapon dynamics, was integral to achieving the desired gameplay experience.
Slo-Mo Integration: Implementing slo-mo required a deep dive into altering game dynamics, time scaling, and user interaction, fostering a better understanding of game physics and user engagement strategies.
3D Modeling and Artistry: The extensive modeling work in Blender taught vital skills in asset creation, environment design, and character modeling, emphasizing the importance of visual appeal in gaming.
Learning Outcomes:
Holistic Game Development Understanding: This project provided a comprehensive overview of the various facets of game development, from coding intricacies to 3D modeling challenges.
Adaptability and Problem-Solving: Overcoming hurdles in modeling and programming demanded adaptive thinking and problem-solving skills, refining my ability to address complex challenges.
Visualization and Creativity: Understanding how to translate ideas into tangible game elements honed visualization skills and fueled creative thinking in designing engaging gameplay.
Conclusion:
The Run and Gun Copy project was not merely about replication; it was a profound learning experience that illuminated the multifaceted nature of game development. It provided practical insights and skills crucial for navigating the dynamic world of game design and creation.
This undertaking fostered an appreciation for the meticulous balance between technical proficiency and artistic creativity required in developing captivating and immersive gaming experiences.
