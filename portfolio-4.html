Mencherz

2021 - 2022
Incytel Games
Unity, Mobile, VFX

Mencherz is a free-to-play mobile game that combines puzzle mechanics with social features, creating a highly engaging experience for players. As the lead Unity developer on this project, I was responsible for implementing core gameplay systems, visual effects, and optimizing performance across a range of mobile devices.

Key Contributions

Led a team of 4 developers to build the game from concept to release. Designed and implemented the match-3 puzzle system with special combo effects that created visually stunning chain reactions. Developed the social interaction system allowing players to compete and collaborate in real-time.

Created custom particle systems and shaders to enhance the visual appeal of game elements. Optimized memory usage and rendering performance to maintain 60fps on mid-range Android devices. Integrated Firebase for analytics and remote configuration to enable live operations.

Technical Highlights

Implemented a custom state machine for game flow management. Developed a modular VFX system that allowed designers to create complex effects without code. Created shaders for various visual effects including glow, distortion and dissolve transitions. Integrated PlayFab for backend services including player data storage and matchmaking.

Results

The game achieved over 100,000 downloads in its first month and maintained a 4.7/5 rating on both App Store and Google Play. Player engagement metrics showed an average session length of 12 minutes with 35% day-7 retention.

Hakem Sho

2020 - Present
Incytel Games
Unity, Multiplayer, UI

Hakem Sho is a social deduction game where players take on roles of judges and participants in a courtroom setting. As the technical lead, I architected the networking solution and implemented the core game systems that enabled up to 12 players to interact in real-time.

Key Contributions

Designed and implemented the custom networking solution using Photon PUN2 to ensure smooth multiplayer interactions. Created the role assignment and game state management systems that handle complex player interactions. Developed the voice chat system with spatial audio features.

Built a comprehensive analytics system to track player behavior and game balance. Optimized network code to reduce bandwidth usage by 40% while maintaining responsiveness. Created custom UI components that adapted to different screen sizes and orientations.

Technical Highlights

Implemented a priority-based messaging system that ensured critical game state updates were delivered without delay. Developed a custom matchmaking system that grouped players by skill level and preferred language. Created tools for the design team to simulate multiplayer scenarios locally for testing purposes.

Results

The game became one of the top social games in its category with thousands of simultaneous players during peak hours. It maintained a 4.8/5 rating with particularly high marks for networking reliability and social features.
