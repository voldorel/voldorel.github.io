Avangard

Avangard is a strategy RPG battle game that combines tactical combat with character progression in a beautifully crafted fantasy world. As the lead developer, I was responsible for the entire development process from concept to release.

Key Features

  • Strategic turn-based combat system with depth and replayability
  • Diverse hero selection with unique abilities and skill trees
  • Resource management and base building elements
  • Stunning 3D graphics optimized for mobile devices
  • Cross-platform compatibility (iOS, Android, PC)
  • Engaging single-player campaign with branching storylines
  • Online PvP battles with ranking system

Technical Highlights

The game was developed using Unity Engine with a focus on performance optimization for mobile devices. I implemented custom shaders for visual effects, developed an efficient AI system for enemy behavior, and created a robust server backend for multiplayer functionality and data persistence.
