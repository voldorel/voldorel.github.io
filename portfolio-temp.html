Cosmic Adventure

Game Development | Unity | C#

×

Cosmic Adventure is a space exploration game built with Unity that features procedurally generated star systems, dynamic economies, and emergent storytelling. Players can explore over 100 unique planets, each with their own ecosystems, civilizations, and mysteries to uncover.

The game features a custom physics engine for realistic orbital mechanics, a branching narrative system with over 50 hours of content, and a dynamic AI system that remembers player actions and adapts the world accordingly.

My role included designing the procedural generation algorithms, implementing the core gameplay systems, and creating the visual style that blends retro sci-fi aesthetics with modern rendering techniques.

Neon Dreams

Interactive Art | TouchDesigner | Projection Mapping

×

Neon Dreams is an interactive art installation that transforms physical spaces into living cyberpunk environments. Using projection mapping and real-time motion tracking, the installation responds to viewers' movements with dynamic visual and audio feedback.

The project was developed using TouchDesigner for real-time graphics processing, depth cameras for motion tracking, and custom shaders to create the distinctive neon aesthetic. The installation has been featured at several digital art festivals and was awarded the "Most Innovative Use of Technology" at the 2021 Digital Art Awards.

My contributions included designing the interactive systems, developing the visual language, and implementing the projection mapping techniques that adapt to different architectural spaces.

Exhibited: March 2021

Puzzle Dimensions

Mobile Game | Unity | iOS/Android

×

Puzzle Dimensions is a mobile game that challenges players to think in multiple dimensions. The core mechanic involves manipulating 2D puzzles in 3D space by rotating the game world to find new perspectives that solve seemingly impossible puzzles.

The game features 120 hand-crafted levels that gradually introduce new mechanics and increase in complexity. It was designed specifically for touch interfaces with intuitive gesture controls and haptic feedback to enhance the spatial reasoning experience.

As the lead developer, I created the custom physics system that handles the dimensional transitions, designed the progression system that teaches players new concepts organically, and optimized the game for performance across mobile devices.

Released: November 2020

VR Museum

Virtual Reality | Unreal Engine | Oculus

×

The VR Museum is an educational virtual reality experience that allows users to explore historical artifacts and artworks in an immersive 3D environment. The project was developed in collaboration with several cultural institutions to make their collections accessible to a global audience.

Key features include photorealistic 3D scans of artifacts, interactive information displays, guided tours by virtual docents, and multiplayer functionality that lets visitors explore together. The experience supports both room-scale VR and seated modes with different interaction schemes for each.

My work on this project included implementing the core VR interaction systems, optimizing high-polygon 3D models for real-time rendering, and developing the networked multiplayer functionality that synchronizes experiences across users.
